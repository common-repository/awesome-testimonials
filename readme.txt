Integrate a testimonial into your WordPress web site.

== Description ==

Awesome Testimonial Plugin for WordPress built to display testimonials, client quotes or reviews with variety styles on any post, page, custom template, widget, etc


> <strong>A FEW FEATURES ARE</strong><br>


* Fade, Crossfade, Scroll, Cover, and Uncover effects
* Widget Ready
* With just a shortcode, create a list page for testimonial 
* Mobile-Friendly and Responsive - Simple to Customize
* Simple to Use 
* Start ratings are available


### Rate The Plugin
	
* [I'd love to hear from you!](https://wordpress.org/support/view/plugin-reviews/awesome-testimonials#postform)



== Installation ==

1. Upload `Awesome_Testimonial.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[pra_testimonial]` in your Pages.
